Bonus codes

1. Avant toutes choses il faut être connecté avec son compte Total battle, si non ça ne fonctionnera pas.

Cliquez sur Lien, puis introduisez vos indentifiants (adresse e-mail et mot de passe). Une fois dans le jeu, revenir sur cette page

2. Pour télécharger le fichier TXT il faut cliquer avec le bouton droite de la souris sur le Lien, puis enregistrer le liens sous...

3. Après avoir téléchargé le fichier il faut renommer l'extention .txt en .bat (les 3 dernières lettres)

exemple : tb-bonus.txt deviendra tb-bonus.bat

4. Double cliquez sur le fichier une fenetre noir devrait s'ouvrir. avec de informations et votre navigateur lancdera environ 40x le jeux dans de nouveaux onglets.

Pour ceux qui veulent avoir un fichier a executer une fois par jour afin de toujours avoir le bon cadeau du jour, je vous laisse faire de meme avec ce fichier Lien

Pour ceux qui ne sont pas très fortiche avec ce qui précède, télécharger le fichier total.zip, aller dans le dossier téléchargements puis ouvrez le fichier total.zip et double cliquez sur ce qui vous intéresse. Lien
